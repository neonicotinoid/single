# Стартовый HTML\CSS проект для вёрстки
Используется npm, gulp, для установки дополнительных библиотек -  bower. 

## Команды
 - *gulp watch* - запускает рабочую среду. Отслеживает изменения в scss, html, js файлах и собирает готовые файлы в директории /build. 
 - *build:css* - одна из итоговых команд для production-сборки. Очищает весь итоговый css-код и компилирует заново. Добавляет префиксы с помощью gulp-autoprefixer; объединяет одинаковые media-запросы; дополнительно генерирует минифицированный файл стилей. 
 - *imgmini* - берёт все изображения из /img/, оптимизирует и складывает в build/img.
